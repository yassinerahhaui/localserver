Custom HTTP Server

A lightweight, event-driven HTTP/1.1 server built with Java NIO

🚀 High Performance

Non-blocking I/O with Java NIO Selector for handling thousands of concurrent connections

⚙️ Configuration

Fully configurable with JSON. Support for multiple ports, routes, and custom error pages

📁 Static Files

Serve static content with proper MIME types and directory handling

🔧 CGI Support

Execute Python and Bash scripts with environment variable support

📤 File Upload

Handle POST requests with file uploads and configurable size limits

🛡️ Error Handling

Comprehensive error pages and graceful handling of edge cases

Quick Start

Compile and run:

javac -d . src/**/*.java && java Main

Server will start on configured ports (default: 8080, 8081)
