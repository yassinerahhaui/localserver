Quick Start Guides

Getting Started with This Server

This server is built with Java NIO for high-performance non-blocking I/O. Here's how to get started:

Build the project:

javac -d bin src/**/*.java

Run the server:

java -cp bin Main config.json

Access the server:

curl http://localhost:8080

Key Features:

  • Non-blocking I/O with Java NIO Selector
  • Configuration-based routing
  • Static file serving with MIME type detection
  • CGI script execution (Python, Bash)
  • Session and cookie management
  • Custom error pages
  • Multi-port and multi-server support
  • Admin dashboard with metrics
